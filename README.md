# homebrew-tap

Homebrew formulae for [HDR](https://github.com/hdresearch) projects.

## Install

```bash
brew tap hdresearch/tap
brew install ziggit
```

## Formulae

| Formula | Description |
|---------|-------------|
| [ziggit](https://github.com/hdresearch/ziggit) | A modern git implementation written in pure Zig — 4-10× faster, 142KB WASM, succinct mode for LLMs |

## Update

```bash
brew update
brew upgrade ziggit
```
